# CC Mine Knowledge

基于 Docusaurus 的个人知识库，适合沉淀文档、面试 QA、项目复盘和博客文章。

## 快速启动

```bash
npm install
npm start
```

默认本地地址：`http://localhost:3000`

## 内容结构

- `docs/`：文档区（含面试 QA）
- `blog/`：博客区
- `static/`：静态资源

## 常用命令

```bash
npm start       # 本地开发
npm run build   # 生产构建
npm run serve   # 本地预览 build 结果
```
