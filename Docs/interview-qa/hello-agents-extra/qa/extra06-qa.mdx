# Extra06 GUI Agent Q&A

对应原文：[Extra06-GUIAgent科普与实战](https://github.com/datawhalechina/hello-agents/blob/main/Extra-Chapter/Extra06-GUIAgent%E7%A7%91%E6%99%AE%E4%B8%8E%E5%AE%9E%E6%88%98.md)

<QAItem question="Q1：GUI Agent 与传统 RPA 的核心差异？">
RPA 依赖固定规则与脚本，GUI Agent 结合视觉感知与大模型推理，能在界面变化时进行一定自适应决策。
</QAItem>

<QAItem question="Q2：GUI Agent 的三层架构是什么？">
感知层（看见界面）、推理层（规划动作）、执行层（点击输入等操作），三者形成闭环控制。
</QAItem>

<QAItem question="Q3：GUI Agent 当前主要局限？">
对复杂页面和异常状态鲁棒性仍有限，时延和成本偏高，跨平台一致性与安全控制仍是工程难点。
</QAItem>
