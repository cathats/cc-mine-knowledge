# Extra05 Agent Skills 与 MCP Q&A

<QAItem question="Q1：Skills 和 MCP 的本质区别是什么？">
MCP 提供标准化工具接口，解决“能调用什么”；Skills 提供任务方法与领域约束，解决“该怎么做”。
</QAItem>

<QAItem question="Q2：什么是渐进式披露？">
按需加载上下文：先元数据，再核心指令，最后脚本和参考资料，减少上下文膨胀与注意力浪费。
</QAItem>

<QAItem question="Q3：高质量 Skill 的关键原则？">
描述精准、单一职责、确定性优先、资源分层清晰，并通过脚本把高风险步骤固化为可复用执行单元。
</QAItem>
