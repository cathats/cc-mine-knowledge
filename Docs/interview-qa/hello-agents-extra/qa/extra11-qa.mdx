# Extra11 WebAgent Q&A

对应原文：[Extra11-WebAgent科普与实战](https://github.com/datawhalechina/hello-agents/blob/main/Extra-Chapter/Extra11-WebAgent%E7%A7%91%E6%99%AE%E4%B8%8E%E5%AE%9E%E6%88%98.md)

<QAItem question="Q1：WebAgent 的核心能力是什么？">
在网页环境中完成感知、导航、表单操作、信息抽取与任务闭环执行，实现“看网页并操作网页”。
</QAItem>

<QAItem question="Q2：WebAgent 难点在哪？">
页面结构动态变化、反爬机制、异步加载、跨站流程和安全权限控制，都会影响稳定执行。
</QAItem>

<QAItem question="Q3：工程上如何提升成功率？">
结合 DOM 与视觉双通道定位、失败重试策略、步骤校验点、任务分解和可观测日志体系。
</QAItem>
