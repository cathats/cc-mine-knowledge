# Extra07 环境配置 Q&A

对应原文：[Extra07-环境配置](https://github.com/datawhalechina/hello-agents/blob/main/Extra-Chapter/Extra07-%E7%8E%AF%E5%A2%83%E9%85%8D%E7%BD%AE.md)

<QAItem question="Q1：Agent 项目环境配置的最小闭环是什么？">
Python 版本确认、虚拟环境、依赖安装、LLM API 与搜索 API 配置、连通性测试、样例脚本跑通。
</QAItem>

<QAItem question="Q2：为什么推荐 .env 管理配置？">
便于本地隔离、避免密钥硬编码、降低多环境切换成本，也更适合团队协作与自动化部署。
</QAItem>

<QAItem question="Q3：排查 API 失败时先看什么？">
先看密钥与 base_url，再看网络连通和模型名，最后检查代码中的环境变量读取与请求参数。
</QAItem>
