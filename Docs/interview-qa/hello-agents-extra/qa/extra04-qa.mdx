# Extra04 Datawhale FAQ Q&A

对应原文：[Extra04-DatawhaleFAQ](https://github.com/datawhalechina/hello-agents/blob/main/Extra-Chapter/Extra04-DatawhaleFAQ.md)

<QAItem question="Q1：Hello-Agents 在框架生态里的定位是什么？">
更偏学习与实践导向，强调多智能体、工具协作与可落地流程，不是单一框架替代，而是工程化方法补充。
</QAItem>

<QAItem question="Q2：A2A/ANP 这类协议为什么重要？">
协议定义 Agent 间通信语义与协作规则，降低异构系统耦合，提升可扩展性和互操作能力。
</QAItem>

<QAItem question="Q3：学习路径如何规划更高效？">
先单 Agent 基础（Prompt、工具调用、RAG），再多 Agent 编排，最后做性能治理与评估闭环。
</QAItem>
