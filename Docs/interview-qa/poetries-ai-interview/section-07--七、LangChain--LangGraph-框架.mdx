<QAHtmlItem question={`140. LangChain 是什么？它真正帮你做了什么？`} html={`<p>LangChain 把"LLM 应用开发的常见动作"封装成可组合的组件：</p>

<QAHtmlItem question={`142. LangGraph 是什么？为什么有了 LangChain 还要它？`} html={`<p>LangChain 的 chain 是<strong>线性管道</strong>，无法表达：</p>
<ul><li>条件分支（这步成功才走下一步）；</li> <li>循环（不达标就回头）；</li> <li>多 Agent 并行 / 协作；</li> <li>显式状态管理。</li></ul>
<p>LangGraph 是<strong>图编排引擎</strong>，节点 = 一个动作（LLM 调用 / 工具调用 / 自定义函数），边 = 转移规则。</p>
<p>适合：</p>
<ul><li>多 Agent 协作；</li> <li>Agentic RAG（要不要再查一次）；</li> <li>复杂工作流（审批、订单、客服路径）；</li> <li>显式可视化整个 Agent 流程。</li></ul>`} />
